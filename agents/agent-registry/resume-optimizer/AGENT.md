# Resume Optimizer Agent

多Agent协作的简历优化系统。输入JD+原始简历，输出针对目标岗位优化的简历。

## 架构

```
用户输入: JD + 简历原文 + 目标岗位
       │
       ▼
┌──────────────┐   ┌──────────────┐
│  JD解析师     │   │  简历分析师    │  ← 并行
│  Agent       │   │  Agent       │
└──────┬───────┘   └──────┬───────┘
       │                  │
       ▼                  ▼
┌──────────────────────────────┐
│        差距分析师 Agent        │
└──────────────┬───────────────┘
               │
               ▼
┌──────────────────────────────┐
│      简历重写师 Agent          │
└──────────────┬───────────────┘
               │
               ▼
┌──────────────────────────────┐
│      数值审核员 Agent          │
└──────────────┬───────────────┘
               │
         ┌─────┴─────┐
         │ 审核通过?  │
         └─────┬─────┘
          No   │   Yes
          ▼         ▼
    [回到重写]   [输出最终简历]
```

## 5个Agent角色

| 角色 | 职责 | 输入 | 输出 |
|------|------|------|------|
| JD解析师 | 提取岗位关键词、技能要求、经验要求 | JD文本 | 结构化需求清单 |
| 简历分析师 | 诊断简历结构、内容、匹配度 | 简历原文 | 简历诊断报告 |
| 差距分析师 | 对比JD需求和简历现状，排优先级 | 需求清单+诊断报告 | 优化方向文档 |
| 简历重写师 | 按STAR法则改写，嵌入关键词 | 优化方向+原简历 | 优化后简历 |
| 数值审核员 | 校验数据合理性（论文/行业数据对比）| 优化后简历 | 审核报告 |

## 可复用Skill模块

| Skill | 功能 | 被谁调用 |
|-------|------|---------|
| keyword_extractor | JD关键词提取+频率统计 | JD解析师 |
| gap_analyzer | 简历-JD匹配度计算 | 差距分析师 |
| star_method_writer | STAR法则改写经验描述 | 简历重写师 |
| data_validator | 数值合理性校验（论文理论值对比）| 数值审核员 |
| resume_formatter | 简历格式化输出 | 简历重写师 |

## 用法

```python
from agents.agent_registry.resume_optimizer import ResumeOptimizer

optimizer = ResumeOptimizer()
result = optimizer.run(
    jd_text="岗位描述文本...",
    resume_text="原始简历文本...",
    target_role="AI Agent算法工程师"
)
```

## 质量闭环

- 数值审核不通过 → 自动回到重写阶段
- 最多重试3次
- 第3次仍不通过 → 标记需人工审核，输出当前最优版本
